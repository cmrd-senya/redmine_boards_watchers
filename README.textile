h2. Redmine boards/wiki/issues watchers management and sticky groups plugin

* Redmine運用上、超絶便利な「Redmine boards/wiki/issues watchers management and sticky groups plugin」 を Redmine3.x上で動作するように変更したものです。
* 動作確認済み環境： Bitnami Redmine 3.2.3-0/3.3.1-0/3.3.2-2
* Redmine 2.x系との互換性はありません。
* 機能詳細やインストール手順などは "本家":https://github.com/previousnext/redmine_boards_watchers を参照してください。
